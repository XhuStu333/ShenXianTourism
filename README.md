# ShenXianTourism
- A group of students' web homework
- An Attempt at a Student's Homework

## 使用
1. 将sql文件导入mysql数据库
2. 在vsc中开启服务器（有nodejs环境的条件下)
- 新建终端
- 输入
  - `cd travelSpot`
  - `npm run start`
  - 出现
  ```
  > travelspot@0.0.0 start
  > node ./bin/www
  ```
  即为启动成功
3. 启用评论中socket功能时需开启两个服务端 
具体命令行：`node socket.js`
5. 打开index.html
